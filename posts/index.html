  • Sizebound C++ demo release

    Dec 12, 2025

    The rewrite of Sizebound in C++ has been quite a wild journey!
    But now it’s finally ready! At least the new demo is ^^

    # What’s new?

    • The whole gameplay has been redone in C++
    • All the models and textures have been redone
    • The level editor has been redone (even if still WIP)

    # Features

    • Added a few more levels
    • Started adding gamepad support

    # Fixes

    • Fixed some bugs
    • Fixed some performance issues
    • Fixed some softlocks

    # What’s next?

    • I will need to get to the story writing phase now.
    • Adding more details in level to follow the story
    • Finishing the coop system
    • Working on the level design for both singleplayer and coop campaigns
    • Adding the modding system (partially done but still very early)

    That also means that I should be able to post a bit more now that I’m done with the C++ rewrite, now I’ll be able to show abit more of the new features and stuff like that as I add them ^^
