# Hey, I’m Flo_12344!

Florent Daerden (aka Flo_12344), I’m a french selft-taught indie game developer working solo on Sizebound, amind-bending puzzle game blending Portal, Superliminal and Q.U.B.E.

# My Journey

I’ve been tinkering with code and 3D art for about 14 years, starting with basic Minecraft mods and Blender/Cinema 4D experiments. In 2018, I pursued a 3D/VFX bachelor’s degree, but programming kept comming back to my mind. School projects forced me to dive deeper in C#, and I fell in love with game dev and programming all over again.

After graduating, I:

  • Joined multiple game jams to sharpen my skills and learn more about time limitation, scope creep and more
  • Made a simple C++ “engine” as many game dev experimenting and learning graphics programing
  • Started working on Sizebound during the GMTK Game Jam 2024 and turned it into a full project after seeing the overwhelmingly positive feedback from other people in the jam

# What I do now

  • Developing Sizebound in C++ in Godot for better performance and because I have more knowledge in C++ then C#
  • Building YACPPM (Yet Another C++ Manager) to make my C++ projects easier to manage the code should soon be available on github
  • Experimenting with 3D tools, graphics programming and more
